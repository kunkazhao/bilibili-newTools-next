# Scripts

Windows launcher scripts are centralized here.

- `start-main.bat`: start main version (frontend `5173`, backend `8000`)
- `stop-main.bat`: stop main version
- `start-main-and-stable.bat`: start both main + stable versions
- `stop-main-and-stable.bat`: stop both main + stable versions
- `start-stable.bat`: start stable version only (frontend `15173`, backend `18000`)
- `start-backend-dev.bat`: start backend dev server only (`8000`)
