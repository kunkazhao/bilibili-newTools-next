- [ ] **选品库分类编辑优化** (2026-02-13)
  - **需求描述**：在选品库页面中，当用户编辑商品分类并将其修改为其他分类时，该商品应立即从当前分类列表中移除，提供即时的视觉反馈。同时在后台静默调用API完成分类更新操作。
  - **具体实现**：
    1. 前端：分类修改确认后，立即从当前列表状态中移除该商品项
    2. 后端：异步调用分类更新接口，无需等待响应即可完成前端操作
    3. 错误处理：如果后端更新失败，需要回滚前端状态并提示用户
  - **用户体验**：避免用户修改分类后仍看到商品停留在原分类列表中的困惑
  - **前置条件**：需要先完成上述分类编辑功能修复
